Langchain: Chatting with PDFs

This AI project enables natural language conversations with PDFs by ingesting links, processing text via embeddings, and enabling interactive user queries. After scraping 1321 links, the project leverages Langchain for text segmentation and Pinecone for efficient storage and retrieval. With an OpenAI model, users can now engage in intuitive chats about processed content.
